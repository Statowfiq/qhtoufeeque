      Steps to create a return from Customer support feature with in BackOffice Hybris 6.3

      3 weeks ago by Admin
      To create a return, you must have access of Customer Support BackOffice. Please follow below steps to enable a user to create a return via customer support office feature of BackOffice.   How to get ... ( Read more )

      To create a return, you must have access of Customer Support BackOffice. Please follow below steps to enable a user to create a return via customer support office feature of BackOffice.

       

      How to get access of CustomerSupport options in BackOffice

      Admin person can assign a user to customersupportadministratorgroup and once you get access then during login it will ask to select one of role as below

       

      Hybris provides below 3 roles which CSR person can select based on their requirements

      1. customersupportmanagerrole

      2. customersupportagentrole

      3. customersupportadministratorrole

       

       

      https://localhost:9002/backoffice/


      Where customersupportadministratorrole have full access of returns functionalities.

      Once click on processed, it will redirect to landing page with default option 'Administration' mode.

       

       

      Select CustomerSupport option to see the returns features


      Click on Create a return to start with creating a return as a CSR person.

       

      ImpEx to update roles and groups

      $lang=en

      UPDATE UserGroup;uid[unique=true];locname[lang=$lang]

       ;customersupportadministratorgroup;"Customer Support Administrator Group"

       

      UPDATE BackofficeRole;uid[unique=true];locname[lang=$lang]

       ;customersupportadministratorrole;"Customer Support Administrator Role"

       ;customersupportagentrole;"Customer Support Agent Role"
